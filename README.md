## 사용 방법

[ghcr Image](https://github.com/so1s/so1s-model-templates/pkgs/container/model-templates)로 배포되어 있어 Base Image로 사용하시는 것이 가능합니다.

Repo Clone을 통한 보일러플레이트를 ML 프로젝트에 연동하셔서 사용하시는 것도 가능합니다.
